# 论文翻译规范参考

## 翻译基本原则

1. **准确性优先** — 忠实于原文的学术含义，不添加、不遗漏
2. **术语一致性** — 同一术语在全文中译法一致
3. **学术风格** — 使用学术论文的中文表达规范
4. **逐句对照** — 保留原文与译文的逐句对应关系

## 翻译格式要求

### 段落级翻译
```markdown
**原文：**
[英文原文段落]

**译文：**
[中文译文段落]
```

### 句子级翻译（适合精细对照）
```markdown
**原文：** [英文原句]

**译文：** [中文译句]
```

## 注意事项

1. **数学公式** — LaTeX 公式保持原样，不做翻译
2. **专有名词** — 首次出现时保留英文并附中文译名，如 "Transformer（变换器）"
3. **参考文献引用** — 保持 `[1]`, `(Smith et al., 2020)` 等引用格式不变
4. **缩写词** — 首次出现时给出全称，如 "CNN（卷积神经网络，Convolutional Neural Network）"
5. **图表引用** — 保留 "Figure 1" / "Table 2" 等原文引用，可附中文 "图1 / 表2"
