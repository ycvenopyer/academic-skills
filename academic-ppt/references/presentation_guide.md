# 组会论文汇报PPT参考指南

## 标准汇报结构（建议12-18张幻灯片）

| 幻灯片 | 内容 | 时长(分钟) |
|--------|------|-----------|
| 标题页 | 论文标题、作者、来源、汇报人 | 0.5 |
| 目录 | 汇报提纲 | 0.5 |
| 研究背景 | 问题背景、研究动机 | 2 |
| 相关工作 | 现有方法及不足 | 2 |
| 核心方法 | 论文提出的方法/模型 | 5 |
| 实验设置 | 数据集、评估指标、参数 | 2 |
| 实验结果 | 定量结果、消融实验 | 3 |
| 结论 | 主要贡献与结论 | 1 |
| 讨论与思考 | 优缺点分析、个人思考 | 2 |
| Q&A | 致谢 | - |

## 每页内容要点

### 标题页
- 论文完整标题（中英文）
- 作者列表、发表会议/期刊、年份
- 汇报人姓名、日期

### 方法页
- 方法概览图（从论文中提取）
- 算法伪代码或关键公式
- 与现有方法的对比示意图

### 实验页
- 数据集说明表
- 主要结果对比表（基线 vs 本文方法）
- 消融实验结果

### 讨论页
- 论文优点（创新性、实验充分等）
- 可能的问题（局限性、未探索方向）
- 对本课题的启发

## 设计建议

1. **图文并茂** — 每页至少包含1张图片或表格
2. **重点突出** — 关键数据用颜色/加粗强调
3. **简约大方** — 避免大段文字，每页不超过8行
4. **一致性** — 全篇使用统一的配色和字体
5. **中文为主** — 关键术语保留英文

## 学术汇报常用配色

- **蓝色系**（标准学术）: 深蓝(#1B3A5C) + 亮蓝(#3B82C4) + 白色
- **绿色系**（自然学科）: 墨绿(#2D6A4F) + 浅绿(#95D5B2) + 白色
- **红色系**（工程学科）: 深红(#8B0000) + 金色(#C99A3B) + 白色
